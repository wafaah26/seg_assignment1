How I Work

I'm currently learning UI design as part of this course at the University of Ottawa. Throughout this semester, I’m exploring key design concepts such as user-centered design, accessibility, and heuristic evaluation. This portfolio will gradually include examples of how I apply these ideas to my projects.

While I’m new to formal UI design, I have a bit of experience working on software development projects. In the Fall semester, I built a project focused on item rentals, and in the Winter semester, I worked on a hotel booking system. These projects helped me understand how design choices impact usability and user experience.

I constantly try to improve my skills by learning from online resources such as GeeksforGeeks, W3Schools, and various YouTube tutorials. I find these platforms helpful for both design theory and implementation tips.

As the semester continues, I’ll be adding more details about my design process and how I apply what I’m learning in each of the upcoming projects.
